📋 Descripción del Proyecto

Gestión de Vuelos es una aplicación de escritorio desarrollada en JavaFX que permite administrar información relacionada con pasajeros y vuelos. La aplicación ofrece funcionalidades para listar, insertar y gestionar pasajeros y sus respectivos viajes, facilitando la gestión eficiente de datos en el ámbito de la aviación.

🛠️ Tecnologías Utilizadas

Java 17: Lenguaje de programación principal para desarrollar la lógica de la aplicación.
JavaFX 17: Biblioteca gráfica para construir interfaces de usuario ricas y responsivas.
FXML: Lenguaje XML utilizado para definir la estructura y diseño de las interfaces de usuario de JavaFX.
Maven: Herramienta de gestión de proyectos y dependencias que facilita la compilación y construcción del proyecto.
Scene Builder: Herramienta visual para diseñar y ajustar archivos FXML de manera intuitiva.
MySQL: Sistema de gestión de bases de datos relacional utilizado para almacenar y gestionar datos de pasajeros y vuelos.
XAMPP: Paquete que incluye MySQL y otros servicios necesarios para el desarrollo local.
Eclipse IDE: Entorno de desarrollo integrado utilizado para escribir, depurar y gestionar el código fuente.

🚀 Funcionalidades Implementadas

1. Gestión de Pasajeros
Insertar Pasajeros: Permite añadir nuevos pasajeros a la base de datos con detalles como nombre, apellido, fecha de nacimiento, tipo y número de documento, email y teléfono.
Listar Pasajeros: Visualiza una tabla de todos los pasajeros almacenados, con columnas detalladas y una interfaz responsiva que se ajusta al tamaño de la ventana.
Buscar Pasajeros: Funcionalidad para buscar pasajeros por número de documento, con soporte para manejar múltiples resultados y selección del pasajero deseado.

2. Gestión de Vuelos
Insertar Vuelos: Permite añadir nuevos vuelos con detalles como código de vuelo, aerolínea, fecha y hora de salida y llegada, origen, destino y estado del vuelo.
Listar Vuelos: Visualiza una tabla de todos los vuelos disponibles, con detalles completos y una interfaz amigable.

3. Gestión de Viajes
Insertar Viajes: Asocia pasajeros con vuelos específicos, asignando asientos y fechas de reserva.
Listar Viajes: Muestra una tabla detallada de todos los viajes, relacionando pasajeros con sus respectivos vuelos.

🔧 Instalación y Configuración

Requisitos Previos
Java 17 o superior instalado en tu sistema.
Maven instalado y configurado.
XAMPP instalado para gestionar el servidor MySQL local.
Eclipse IDE (u otro IDE de tu preferencia) para importar y trabajar con el proyecto.
